### Checklist Before Running
- [ ] Python 3.8+ installed
- [ ] Ollama installed and running (`ollama serve`)
- [ ] AI model downloaded (`ollama pull codellama:13b-instruct`)
- [ ] 8GB+ RAM available
- [ ] 20GB free disk space
- [ ] Read at least README_AUTOMATION.md or QUICK_REFERENCE.md
- [ ] Have 2-3 continuous hours available

### To Get Started

**Step 1: Install Dependencies (5 min)**
```bash
pip install -r automation/requirements.txt
```

**Step 2: Start Ollama (in separate terminal)**
```bash
ollama serve
```

**Step 3: Download Model (10-15 min)**
```bash
ollama pull codellama:13b-instruct
```

**Step 4: Run Automation (90-135 min)**
```bash
automation\run_chain.bat
```

**Step 5: Monitor Progress** (While executing)
```bash
Get-Content automation/logs/chain_execution.log -Tail 20 -Wait
```

**Step 6: Verify Completion** (After done)
```bash
Get-Content automation/logs/chain_report.json
flutter analyze
```
