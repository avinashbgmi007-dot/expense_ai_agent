## 🔧 CUSTOMIZATION OPTIONS

Before running, you can customize:

### Change AI Model
Edit `automation/chain_config.yaml`:
```yaml
ai_config:
  model: "mistral:7b"  # Faster but less accurate
  # OR
  model: "codellama:34b"  # More accurate (needs 16GB RAM)
```

### Change Timeout
Edit `automation/chain_config.yaml`:
```yaml
ai_config:
  timeout: 600  # 10 minutes instead of 5
```

### Skip Phases
Comment out phases in `automation/chain_config.yaml`:
```yaml
# phase_2:  # Skip categorization
#   ...
```

### Increase Retries
Edit task retry count in `automation/chain_config.yaml`:
```yaml
retry_count: 5  # 3 retries → 5 retries
```
